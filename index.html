Kompetensi Belajar CP & ATP 🎯

🎯 Capaian Pembelajaran (CP)

Peserta didik mampu mengenali berbagai perangkat keras komputer di sekitarnya, mengidentifikasi perbedaan ukuran gawai, membedakan fungsi komponen input dan output data, serta mendemonstrasikan kebiasaan merawat gawai secara aman.

📌 Tujuan Pembelajaran (TP)

  • Mengurutkan 5 tingkatan dimensi gawai komputer secara presisi.
  • Memisahkan gerbang piranti Input pengetikan dan Output cetak visual.
  • Memilih aksi protektif higienis harian untuk menjaga keawetan piranti keras.
